# 🏛️ ViraasatAR
### AR-Based Cultural Heritage Preservation Platform  

**Smart India Hackathon 2025**  
**Problem Statement ID:** SIH25052  
**Theme:** Heritage & Culture  

---

## 📌 Overview  

**AR Cultural Hub** is a web-based Augmented Reality platform designed to digitally preserve and promote India's cultural heritage.  

Users can scan QR codes to instantly view immersive 3D models of monuments, listen to multilingual audio narrations, and discover local cultural events — all optimized for low-bandwidth rural access.

---

## 🚀 Features  

- 🔲 **QR Code AR Activation** — Scan to instantly launch 3D heritage models  
- 🌐 **Multilingual UI** — Regional language support via React i18n  
- 🔊 **AI-Generated Audio Narration** — Powered by Murf.ai for multilingual storytelling  
- 📅 **Cultural Events Discovery** — City-wise event listing and booking integration  
- 📶 **Rural-First Design** — Optimized for low-bandwidth with compressed glTF/WebP assets  
- ☁️ **CDN Delivery** — Fast loading of 3D models and audio across regions  

---

## 🛠️ Tech Stack  

| Layer | Technologies |
|-------|-------------|
| **AR & Frontend** | 8th Wall, ARCore, React.js, HTML5 |
| **Multilingual UI** | React i18n |
| **Audio & Content** | Murf.ai, HTML5 Web Audio, glTF, WebP |
| **Backend** | Headless CMS, Firebase / AWS |
| **Delivery** | CDN |

---

## 🏗️ Architecture  
User → Scans QR Code
→ Initiates AR Session (8th Wall + ARCore)
→ React.js UI with i18n loads
→ Requests 3D models / audio / UI text
→ Served via CDN from Firebase/AWS
→ Headless CMS manages content updates


---

## 💡 Problem It Solves  

| Challenge | Our Solution |
|------------|--------------|
| Heritage sites decaying without digital record | AR-based 3D preservation |
| Rural communities lacking cultural access | Low-bandwidth optimized delivery |
| Youth disengagement from culture | Immersive AR + gamified exploration |
| Language barriers | Multilingual audio & UI |
| Limited heritage data | Open-source + future ASI collaboration |
